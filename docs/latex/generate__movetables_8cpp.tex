\doxysubsection{Detailed Description}
This file generates {\ttfamily \doxylink{movetables_8cpp}{movetables.\+cpp}}. It pre-\/calculates the possible moves for a given position to speed up move generation in the search.

Computes one ray of a rooks movement. The pair {\ttfamily (add\+File, add\+Rank)} describes the direction of the ray\+:


\begin{DoxyItemize}
\item {\ttfamily (+1, 0)} = rook goes to the right
\item {\ttfamily (-\/1, 0)} = rook goes to the right
\item {\ttfamily (0, +1)} = rook goes to the top
\item {\ttfamily (0, -\/1)} = rook goes to the bottom
\end{DoxyItemize}

Other values should not be uses.
